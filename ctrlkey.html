A Tecla CTRL

A tecla CTRL (Control) é usada em combinação com outras teclas para executar funções específicas em programas de computador. Além disso, ela é muito usada para acessar atalhos de teclado em programas e sistemas operacionais

Algumas combinações comuns de teclas com CTRL incluem:
CTRL+C : para copiar
CTRL+V : para colar
CTRL+Z : para desfazer a última ação
